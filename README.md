## How to Update

### Edit content
1. Open the relevant `.html` file in any text editor
2. Make changes and save
3. Upload the updated file to this GitHub repo (replacing the old one)
4. Changes go live within ~1 minute

### Add a new project
1. Copy an existing `project-N.html` as your starting template
2. Update the title, tags, product link, cover image path, and all section text
3. Prepare a cover image and name it `project-N-cover.jpg`
4. Add a new card in `index.html` inside the `.projects-grid` div
5. Upload both the new HTML file and cover image to the repo

### Update profile photo
Replace `avatar.jpg` with a new file of the same name (square crop recommended, min 200Ã—200px).
